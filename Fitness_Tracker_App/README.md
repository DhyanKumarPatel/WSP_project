# Fitness Tracker App

A **Fitness Tracker App** built with **Vue.js**.

## Features
- Track daily workouts
- Log fitness progress
- View activity history
- Simple and responsive UI

## Tech Stack
- Vue.js
- JavaScript
- HTML/CSS

## Project Setup
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
