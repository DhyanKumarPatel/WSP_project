# FitTrack Pro - Fitness Tracker Application

A comprehensive fitness tracking and community platform built with Vue 3, where users can log their daily workouts, track their progress, and connect with friends.

## 🌐 Live Site

**[FitTrack Pro - Live Application](https://fittrack-pro-oyxf.onrender.com)**

Visit the live site to experience the application in action!

## 📋 About the App

FitTrack Pro is a modern fitness tracking application that enables users to:

- **Track Activities**: Log various types of exercises (Running, Cycling, Swimming, Walking, etc.) with duration and calorie metrics
- **Monitor Progress**: View comprehensive statistics and activity summaries
- **Activity Breakdown**: Visualize activity distribution through charts
- **Friend Network**: Connect with other users and view their activities
- **User Profiles**: Manage personal information and fitness goals
- **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices
